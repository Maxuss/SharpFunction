BossbarSet  set,

)

Compile command as "/bossbar set {params}"

Parameters
idID-name of boss bar
setType of data (T) to set to boss bar
extraExtra data, depending on set
  1. BossbarSet.Color > color string (BossbarColor fields are recommended)
  2. BossbarSet.MaxValue > max value 32 bit integer
  3. BossbarSet.Name > show name string
  4. BossbarSet.Players > selector EntitySelector
  5. BossbarSet.Style > style string. Allowed values: (notched_6|notched_10|notched_12|notched_20|progress)
  6. BossbarSet.Value > value short. Value should be less than 100 and more than 0
  7. BossbarSet.Visible > visibility bool.
